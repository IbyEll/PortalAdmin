Applicazione step-by-step

  1. Leggi Jira — parent, subtask, orphan, key invalide (read-only).
  2. Analizza repo — gap analysis; Stato repo + Subtask vs repo.
  3. Aggiorna story — description parent allineata al repo.
  4. Aggiorna subtask — description per ogni subtask.
