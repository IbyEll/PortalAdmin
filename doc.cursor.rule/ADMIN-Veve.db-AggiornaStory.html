Veve DB — Aggiorna Story

Trigger

aggiorna story db {KEY} · dopo Leggi DB + Analizza repo · veve db {KEY} step 3

Obiettivo

Merge description veve sul parent {KEY} — non editJiraIssue MCP.

Template

.cursor/templates/veve/veve-story-parent.md — compila da cache + gap analysis + elenco subtask (anche appena create).

Sezioni e checkbox AC/DoD (B poi A): come ADMIN-Veve-AggiornaStory.mdc.

Scrittura DB

  1. Leggi riga parent da jira_issue (ultimo sync_run success).
  2. Compila veveDescription dal template (tabella Stato repo solo ✅/⚠️/❌).
  3. Scrivi raw_fields su jira_issue_wip (workflow) o jira_issue (CREA Matrix).
