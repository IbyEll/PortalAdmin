Step 7 — Chiusura locale (solo WIP)

Prima (step 7 includeva PR)

close-story.mjs + parent WIP + PR nello stesso step agente.

Adesso
  1. Verifica tutte le subtask is_done = true in WIP.
  2. Gap test (solo chiudi, non chiudi fast).
  3. Update parent WIP: is_done, status=Fatto, raw_fields.awaitingPush=true.
  4. END-B con PR: — — niente push, niente Jira.
  5. Ritorno backlog — UI mostra PUSH.

Niente close-story.mjs · niente Jira live in step 7.

Step 8 — PUSH (Jira da WIP + PR)

Solo dal cruscotto, quando l'utente clicca PUSH sulla riga (post step 7).

  1. Prerequisito — parent WIP con is_done=true e awaitingPush=true.
  2. Leggi WIP — parent + subtask; contenuto da raw_fields.
  3. Sync Jira live — da WIP verso Atlassian:
    • Subtask: description + transizione Fatto (ok-chiudi-subtask.md)
    • Parent: description + transizione Fatto (chiudi-parent.md)
    Task «Jira Auto» o MCP — unico step con Jira.
  4. Git / PR / catalogo: 
    node admin.portal.JiraCORE/jiraCORE.close.story.mjs --key {KEY}
  5. Update WIP — prUrl, jiraSyncedAt, awaitingPush=false.
