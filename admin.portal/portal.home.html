Cos'è PortalAdmin

PortalAdmin è il repository tooling che ospita il cruscotto dev: health stack, esecuzione testScript, backlog Jira, working plan e report HTML/JSON.

  • Overlay PROJECT_{PRJ_NAME} — config, discovery e catalogo test per product
  • API dashboard per overlay — porta dedicata (PRJ_DASHBOARD_PORT, es. JustLastOne :3999, Admin :3998)
  • Segnali Jira ADMIN + product (es. JLO) da portal.config

Scegli un progetto a destra, poi Istanzia per scrivere .env, preparare DB cruscotto e (se configurato) il prepare del monorepo product. Ogni progetto ha una porta dedicata — puoi tenere più cruscotti aperti insieme. Dopo Istanzia, usa Avvia / Kill sulla card o Apri cruscotto per il browser. La HOME resta su :3990 (npm run admin:home).
